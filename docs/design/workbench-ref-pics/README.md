# Workbench Reference Pictures

These images are visual references for [Workbench UI/UX Spec](../workbench-ui-ux-spec.md). They support the current stage of the product only: play, recall, and analysis/review-board workbench UI.

- [Recall Mode Workbench](recall-mode-workbench.png): recall mode layout, green accent, recall task panel, feedback panel, and lightweight variation tree.
- [Analysis Mode Workbench](analysis-mode-workbench.png): analysis/review-board layout, purple accent, analysis flow, notes, AI analysis, variation tree, and snapshot comparison.

The product-wide blueprint remains [Gabaki / Sabaki Training System PRD](../../product/gabaki-training-prd.md). These images are not a complete UI specification for Problem Editor, Problem Mode, Punishment Problem, or Review / spaced repetition pages.
